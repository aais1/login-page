##Simple Responsive Login Page
This repository features a straightforward responsive login page created using HTML and CSS. The project demonstrates the development of a static login page that adjusts well to different screen sizes, making it suitable for various devices.

1.Clone the repository to your local machine:

git clone https://https://github.com/aais1/login-page

2.Navigate to the project directory:

cd login-page

3.Open the index.html file in a web browser to view the responsive login page

##Contribution
Your contributions are not required for this simple project. Feel free to use it as a basic example for creating responsive web pages.
